# Quick-Commerce Market Entry: Should FreshGo Retail Enter Tier-2 India?

**Case type:** Market Entry Analysis
**Client (disguised, fictional):** FreshGo Retail — a national grocery chain
**Question:** Should FreshGo Retail launch a quick-commerce vertical in
Tier-2 Indian cities?

## Files

- [`FreshGo_TierII_Market_Entry.pptx`](./FreshGo_TierII_Market_Entry.pptx) —
  11-slide case deck: market sizing, competitive landscape, unit economics,
  financial projection, risks, and phased recommendation
- [`FreshGo_TierII_Market_Entry_preview.pdf`](./FreshGo_TierII_Market_Entry_preview.pdf) —
  same deck as a PDF, viewable directly in GitHub
- [`FreshGo_TierII_Entry_Model.xlsx`](./FreshGo_TierII_Entry_Model.xlsx) —

## Recommendation (governing thought)

Enter selectively: pilot a 10-store Wave 1 in Tier-2 cities with strong
density and income proxies, gate Wave 2 expansion on stores hitting a
500+ orders/day throughput target by month 12, rather than a blanket rollout.

## Key figures used

- India quick-commerce market: ~$5.5–6.8B today, forecast to reach ~$30B by
  FY2030, growing 40–45% annually
- Dark-store breakeven throughput: metros ~1,200–1,500 orders/day vs.
  non-metro ~850 orders/day (Redseer)
- Modeled single-store payback: ~13 months, at a ₹550 AOV / ₹54 contribution
  margin per order
